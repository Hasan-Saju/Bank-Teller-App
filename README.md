# Bank-Teller-App

A tool for bank employees to manage customer accounts and cards, conduct transactions, and detect fraud with machine learning support.

## Tech Stack

-   **Backend:**
    -   Django (Server)
    -   SQLite (Database)
    -   Postman (API Testing)
-   **Machine Learning:**
    -   TensorFlow
    -   Keras
    -   PyTorch
-   **Frontend:**
    -   ReactJS (Client)
    -   Tailwind
